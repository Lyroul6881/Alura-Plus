# 🎬 Alura Plus

Projeto desenvolvido durante o curso de HTML e CSS, focado na prática de estruturação com HTML5 e estilização avançada com CSS3.

📌 **Sobre o projeto**
Este projeto simula uma página de venda de combos de streaming, aplicando conceitos de posicionamento, responsividade e organização de código. Desenvolvido como parte da minha jornada na faculdade de Sistemas de Informação.

🚀 **Funcionalidades**
* Estruturação semântica completa.
* Estilização com Flexbox.
* Design responsivo para diferentes telas.
* Botões interativos com efeitos de hover.

📂 **Estrutura do projeto**
alura-plus/
├── img/          # Imagens e ícones do projeto
├── index.html    # Estrutura principal
└── styles.css    # Estilos e design
